# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

To report a security vulnerability, please email **ssm@leonardofaoro.com**.

- Do **not** open a public issue for security vulnerabilities.
- Expect an initial response within 48 hours.
- Please include steps to reproduce and any relevant details.
